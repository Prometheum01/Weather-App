## Tech and Libs

**State Managment:** Provider.

**Cache Managment:** Shared Preferences.

**Network:** Dio.

**Connection:** InternetConnectionChecker.

**Location:** Geolocator.

**Animation:** Lottie.

**Other View Libs:** Flutter Svg, Avesome Snackbar Content.
